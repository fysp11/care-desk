# Repo-Local Skills

## Purpose

These are lightweight skill definitions for this repository. They are not
installed global Codex skills; they are local reusable procedures that agents
can read when the task matches.

## Skills

| Skill | Use When |
|---|---|
| [patient-management-case](patient-management-case/SKILL.md) | Planning, implementing, or verifying the case challenge slice. |
| [latest-library-guard](latest-library-guard/SKILL.md) | Using challenge-named libraries or reviewing generated code for stale APIs. |

## Use Policy

Read only the skill that matches the current task. Keep context narrow and use
the linked docs for details instead of copying large sections into prompts.
